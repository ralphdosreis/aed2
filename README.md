# 📚 Algoritmos e Estruturas de Dados 2 (AED2) - UFG

Este repositório contém os exercícios e projetos desenvolvidos durante a disciplina de **Algoritmos e Estruturas de Dados 2**, ministrada no curso de Ciência da Computação da **Universidade Federal de Goiás (UFG)**.

O objetivo principal é o estudo de técnicas avançadas de programação, análise de complexidade, manipulação de arquivos e estruturas de dados.
